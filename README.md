# security-_script
关于安全学习中开发的一些脚本扫描器之类


——————————————————————————————————————————————————————
security中login为对万能验证码的登录界面的注入点检测脚本
          blind为时间型盲注检测脚本
          baidu_orc为通过百度智能云对验证码进行调库进行爆破识别验证码
——————————————————————————————————————————————————————
network中 xssscan为对单一注入类型的xss跨站脚本攻击的xss扫描器
——————————————————————————————————————————————————————
dict中password是一个密码字典
      xss-payload为少量的xss攻击测试用例的payload
